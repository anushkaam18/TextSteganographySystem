# TextSteganographySystem
A simple Java-based application to hide and retrieve secret messages using file handling and string processing.

# Features
* Encodes a secret message into a file
* Decodes and retrieves the hidden message
* Ensures cover text remains unchanged
* Supports multiple test inputs through menu-driven execution

# Technologies Used
* Java
* File Handling (FileReader, FileWriter, BufferedReader)
* String Manipulation
* Object-Oriented Programming
